## Resumen Ejecutivo

**Mr. Studio Tattoo** es un estudio de tatuajes premium ubicado en Miami (Calle Ocho, Little Havana) con una reputación excepcional en realismo y fine line. La web debe reflejar lujo, precisión artística y la energía urbana de Miami. El presupuesto de $25,000 sugiere una experiencia de alto nivel con animaciones sofisticadas y diseño premium.

**Objetivo:** Crear una página web que sea un portafolio digital de calidad museo, que atraiga clientes de alto valor y comunique la excelencia artística del estudio.

## Animaciones y Efectos 3D

### 1. Hero Section Animations

**Máquina Flotante:**
- Rotación suave en 3D (eje Y) - loop infinito
- Movimiento vertical sutil (bobbing effect)
- Partículas de tinta animadas alrededor
- Glow effect que pulsa
- Al scroll: parallax con velocidad diferente

**Implementación:**
```javascript
// Pseudo-código para Lovable
@keyframes float {
  0%, 100% { transform: translateY(0px) rotateY(0deg); }
  50% { transform: translateY(-20px) rotateY(5deg); }
}

@keyframes particleFloat {
  0% { opacity: 0; transform: translate(0, 0); }
  50% { opacity: 1; }
  100% { opacity: 0; transform: translate(var(--tx), var(--ty)); }
}
```

### 2. Scroll-Triggered Animations

**Fade-in on Scroll:**
- Elementos aparecen suavemente al entrar en viewport
- Usar Intersection Observer API

**Counter Animation:**
- Números de estadísticas cuentan hacia arriba
- Trigger: cuando la sección es visible

**Parallax Scroll:**
- Fondos se mueven a velocidad diferente
- Crea profundidad visual

### 3. Hover Effects

**Tarjetas de Servicios:**
- Elevación (box-shadow aumenta)
- Color de acento aparece
- Transición suave (0.3s)

**Botones:**
- Glow effect al hover
- Cambio de color a dorado
- Pequeña escala (1.05x)

**Imágenes de Galería:**
- Zoom suave (1.1x)
- Overlay oscuro aparece
- Ícono de expansión

### 4. Efectos de Partículas

**Ink Particles:**
- Pequeñas partículas flotantes en hero
- Movimiento aleatorio
- Opacidad variable
- Colores: púrpura, azul, dorado

**Implementación:** Usar Three.js o Spline para 3D avanzado.

## Recomendaciones Técnicas para Lovable

### Performance

1. **Optimización de Imágenes:**
   - Usar WebP con fallback a JPG
   - Lazy loading para galería
   - Responsive images (srcset)

2. **Animaciones:**
   - Usar CSS transforms (GPU-accelerated)
   - Evitar animaciones en scroll excesivas
   - Throttle/debounce eventos

3. **Carga:**
   - Precargar imágenes hero
   - Minificar CSS/JS
   - Usar CDN para assets

### SEO

1. **Meta Tags:**
   - Title: "Mr. Studio Tattoo | Miami's Premier Tattoo Studio"
   - Description: "Premium tattoo studio in Miami specializing in realistic and fine line tattoos. Book your session today."

2. **Estructura:**
   - H1 único por página
   - Alt text descriptivo en imágenes
   - URLs amigables

3. **Schema Markup:**
   - LocalBusiness schema
   - Reviews schema
   - OpeningHours schema

### Responsividad

- **Desktop:** Diseño completo con animaciones
- **Tablet:** Ajustes de grid (2-3 columnas)
- **Mobile:** Versión simplificada, CTA flotante
